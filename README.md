# Body Fat Tracker (PWA)

Локальный трекер процента жира по формуле US Navy. Поддерживает три аккаунта с раздельной историей, работает офлайн как PWA.

## Аккаунты по умолчанию
- Пользователи: Bohdan, Konstantin, Izabella
- Пароль для всех: 1234

## Функции
- Переключение формулы (мужчина/женщина) и учёт бёдер для женской формулы.
- Сохранение истории и последнего результата в `localStorage` по каждому аккаунту отдельно.
- График динамики по сохранённым точкам (до 24 последних).
- Удаление отдельных записей и очистка истории.
- PWA: кеширование через service worker, офлайн-доступ.

## Структура
- `index.html` — разметка.
- `style.css` — стили.
- `app.js` — логика, хранение данных, график.
- `service-worker.js` — кеширование статических файлов.
- `manifest.json` — манифест PWA.

## Запуск
Статический проект. Откройте `index.html` в браузере или поднимите любой статический сервер (например, `npx serve .`). Для корректной работы service worker и PWA предпочтителен запуск через http(s), а не `file://`.

## Обновление статики
При изменении `style.css` или `app.js` обновите версию кеша в `service-worker.js` (константа `CACHE` или список `ASSETS`), затем перезапустите / перерегистрируйте service worker или сделайте hard refresh.
